My Scientific Values

Transparency is necessary for reproducible science. I encourage that decisions, methods and deviations are clear and transparent in a workflow from planning phase, to data retrieval and analysis, to final output.

Open by Default

All code and data are published under an open-source license. There are a few exceptions.

Robust Methods
