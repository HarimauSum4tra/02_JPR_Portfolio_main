🔧 Step-by-Step Reproduction


# Step 1: Clone the repository
git clone https://github.com/HarimauSum4tra/[PROJECT_REPO].git
cd [PROJECT_REPO]

# Step 2: Set up R environment (using renv)
Rscript -e "install.packages('renv')"
Rscript -e "renv::restore()"

# Step 3: Run the analysis
Rscript scripts/01_download_data.R
Rscript scripts/02_clean_data.R
Rscript scripts/03_analysis.R
Rscript scripts/04_visualisation.R

# Step 4: Generate report
Rscript -e "rmarkdown::render('report.Rmd')"
